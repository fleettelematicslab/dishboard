# Dishboard 🛰️

**Starlink terminal monitoring for businesses.**
Track signal quality, downtime, and obstructions across all your sites — in one clean dashboard.

## Why Dishboard?

Managing multiple Starlink terminals across construction sites, farms, or vessels is painful.
The official Starlink app only shows one terminal at a time.
Dishboard gives you a single view across your entire fleet.

## Features

- Real-time terminal status (Online / Degraded / Offline)
- Signal quality (SNR) 24-hour trend chart
- Obstruction map visualization
- Downtime alerts
- Multi-site unified monitoring

## Local Development

```bash
npm install
npm start
```

## Deploy to Vercel

1. Push this repo to GitHub
2. Go to vercel.com → "Add New Project"
3. Select this repo → Deploy
4. Done!

## Tech Stack

- React 18 + Recharts + Lucide React
- IBM Plex Mono + Syne fonts
